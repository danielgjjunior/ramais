Documentação de Funcionários

Criar um funcionário:

Rota: POST /funcionarios

Corpo da requisição (exemplo em JSON):

{
  "cidadeId": 1,
  "nome": "João",
  "ramal": "123",
  "telefone_corporativo": "987654321",
  "funcaoId": 1,
  "email": "joao@example.com",
  "Skype": "joao.skype"
}

Obter todos os funcionários:

Rota: GET /funcionarios

Obter um funcionário por ID (substitua :id pelo ID real do funcionário):

Rota: GET /funcionarios/:id

Atualizar um funcionário por ID (substitua :id pelo ID real do funcionário):

Rota: PUT /funcionarios/:id

Corpo da requisição (exemplo em JSON para atualização):

{
  "ramal": "456",
  "telefone_corporativo": "555555555",
  "Skype": "joao.skype2"
}

Deletar um funcionário por ID (substitua :id pelo ID real do funcionário):

Rota: DELETE /funcionarios/:id

Criar uma cidade:

Rota: POST /cidades

Corpo da requisição (exemplo em JSON):

{
  "nome": "São Paulo",
  "estadoId": 1
}

Obter todas as cidades:

Rota: GET /cidades

Obter uma cidade por ID (substitua :id pelo ID real da cidade):

Rota: GET /cidades/:id

Atualizar uma cidade por ID (substitua :id pelo ID real da cidade):

Rota: PUT /cidades/:id

Corpo da requisição (exemplo em JSON para atualização):

{
  "nome": "Rio de Janeiro",
  "estadoId": 2
}

Deletar uma cidade por ID (substitua :id pelo ID real da cidade):

Rota: DELETE /cidades/:id
